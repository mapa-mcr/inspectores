Vigencia del catastro

Este parcelario corresponde al relevamiento 2022–2023. No refleja subdivisiones, cambios de titularidad ni obras posteriores a esa fecha.

Para actualizarlo: reemplazar los archivos de data/shards/ y data/index.json por una exportación nueva del sistema catastral. Cuando exista API de MAGIC/APEX, este paso se automatiza y esta pantalla puede mostrar la fecha de última sincronización real.

Datos de propietario / domicilio

Se ingresan a mano por el inspector en el momento del acta. Este prototipo no está conectado al padrón de contribuyentes — esa conexión requiere acceso institucional a MAGIC/APEX, que queda pendiente de gestión con Sistemas.

Numeración de actas

Formato APP-AAAA-000000, correlativo por dispositivo mientras no hay sincronización con un servidor central. Si se usan varios teléfonos en simultáneo, dos actas de dispositivos distintos podrían compartir número hasta sincronizar — el servidor real debería asignar el número definitivo al recibir el acta.

Prototipo — alcance
